How to build a RingCentral App with Ringcentral Widgets

Start Tutorial

Prerequisites

  • Install Node.js with version >= 8
  • Create a RingCentral developer free account
  • Create a new RingCentral app with platform type - "Browser Based"

Initialize project by Generator Ringcentral Widgets

$ mkdir ringcentral-widgets-demo
$ cd ringcentral-widgets-demo
$ yo ringcentral-widgets

Update .env

Start development server

yarn start

Visit on http://localhost:8080 in browser
